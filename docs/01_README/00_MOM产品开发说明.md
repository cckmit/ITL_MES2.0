# MOM产品模块介绍
产品模块结构如下：

``` xml
├── iap-auth -- 授权服务提供
├── iap-common -- 常用工具封装包
├── iap-gateway -- Spring Cloud 网关
├── iap-ops -- 运维中心
├── iap-service -- 业务模块
├    ├── iap-attachement -- 附件服务 
├    ├── iap-im -- 即时通讯服务 
├    ├── iap-notice -- 消息中心 
├    ├── iap-system -- 系统模块 
├    ├── iap-workflow -- 工作流模块 
├    ├── iap-xxl-job-admin -- 任务调度

├    ├── iap-mes -- mes基础功能 
├    ├── mes-andon -- 安灯管理模块 
├    ├── mes-md -- mes主数据管理模块
├    ├── mes-me -- mes生产执行模块 
├    ├── mes-pp -- mes计划排程模块
├    ├── mes-qm -- mes质量管理模块 


```
